## httpモジュール実装について
Node.jsのhttpモジュールは
TCP.server と events.EventEmitter を継承したもの

追加されているのは
    requestイベント
    IncomingMessageのオブジェクト2種類
        http.ServerRequest(req) と http.ServerResponse(res)
        こいつらがリクエストボディやリクエストヘッダを返すメソッド(writeHeadなど)を持っている

よって、上記のものを実装すれば実現はできる(でも無理だった)
